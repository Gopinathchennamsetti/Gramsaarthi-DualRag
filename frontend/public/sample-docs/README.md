# Sample Documents for RAG Demo

Use these folders to organize downloadable demo documents and map them to ingestion indexes.

- `schemes_index/` -> upload into index name `schemes_index`
- `citizen_faq_index/` -> upload into index name `citizen_faq_index`

Recommended naming:

- Generic briefs: `scheme_generic_<scheme-slug>.pdf`
- FAQ docs: `scheme_faq_<scheme-slug>.pdf`

Example slugs:

- `jal-jeevan-mission`
- `pm-kisan`
- `pmfby`
